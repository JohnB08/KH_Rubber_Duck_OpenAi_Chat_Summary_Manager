Privacy Policy

Effective Date: August 5, 2025

This Privacy Policy explains how data is collected, stored, and used by the Chat API hosted at kanjiteacher.com. This API enables the generation, retrieval, and storage of chat sessions and summaries.

1. Information We Collect

Through use of the API, the following information may be collected:

  • Chat ID (chatId): A unique identifier generated for each chat session.
  • Chat Summaries: Text summaries of user chat sessions, along with a timestamp.
  • Raw JSON Blobs: The raw structure of chat summaries submitted to the API.
  • Chat Histories: A list of past chats associated with a given Chat ID.

2. How We Use Your Information

Information collected via the API is used to:

  • Generate and manage chat sessions.
  • Allow users to retrieve past summaries and histories.
  • Improve the performance and functionality of the service.
  • Perform diagnostic checks using the /health endpoint.

3. Data Retention

Data submitted to the API is stored securely and retained only as long as needed to fulfill the purpose of chat summarization and history retrieval. Users can request deletion by contacting the site administrator.

4. Data Sharing

We do not sell or share your personal data with third parties. However, we may disclose data if required by law or to enforce our service policies.

5. Security

All data is transmitted over HTTPS and stored using appropriate technical safeguards. However, no system is completely secure. Users are encouraged to avoid including sensitive personal data in chat content.

6. Access and Control

You can retrieve your summaries and chat history using your chatId via the API endpoints:

  • GET /chat_summary?id=<chatId>
  • GET /chat_history?id=<chatId>

7. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date.

8. Contact

If you have questions or concerns about this policy, please contact: support@kanjiteacher.com
